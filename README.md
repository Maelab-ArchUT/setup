# setup — 研究室 環境構築ドキュメント

Mae 研究室で開発・解析を始めるための環境構築マニュアルを置くリポジトリです。

## 収録ドキュメント

- [WSL2 + Claude Code + GitHub 環境構築マニュアル](WSL_ClaudeCode_GitHub_セットアップ.md)
  新規 Windows 11 PC からゼロで WSL2・Claude Code・GitHub CLI を立ち上げる完全手順。**まだ何も入っていない PC でも、ブラウザでこの文書を読みながら上から順にコマンドをコピーして実行すれば環境が整います。**

## 進め方

1. まず上記マニュアルをブラウザで読みながら手作業で環境を構築する (この段階では Claude Code はまだ無い)。
2. 構築が完了したら、以降は各ツールのリポジトリを clone し、Claude Code を起動して対話しながら作業する。各リポジトリのルートにある `CLAUDE.md` がそのツールの使い方を説明します。
