# CLAUDE.md — setup

このリポジトリで Claude Code が作業するための参照情報。
人間向けの概要・進め方は [README.md](README.md) を見ること。

## 主編集者

竹中（GitHub: `03210083` / `among-take`）。本リポジトリは竹中の管轄。
研究室メンバーが環境構築のために読む（参照する）のは自由だが、ドキュメントの変更は
主編集者の管轄。別の人が編集する場合は主編集者と調整すること。

## 公開設定（重要）

**本リポジトリは Public。** `Maelab-ArchUT` Org 内では唯一の公開リポジトリであり、
内容が一般公開される前提で扱うこと。編集時は次を厳守する:

- 個人情報（実名・メールアドレス・住所等）、認証情報（APIキー・トークン・パスワード）、
  研究データや非公開ノウハウを一切含めない。固有値は必ずプレースホルダにする。
- 他の研究室リポジトリは Private。本リポジトリから他リポジトリの内部情報（パス構成・
  実装詳細・データ）を引用・転記しない。Org 名と「管理者に招待依頼」の導線に留める。
- 公開状態の変更（Private 化など）は主編集者の判断で行う。

## このリポジトリの役割

研究室の環境構築ドキュメント置き場。新メンバーが WSL2 + Claude Code + GitHub を
ゼロから立ち上げるためのマニュアル（`WSL_ClaudeCode_GitHub_セットアップ.md`）を収録。

このマニュアルは「まだ Claude Code が無い人がブラウザで読みながら手作業で構築する」
ことを前提とする。したがって編集時は次を守る:

- コマンドはコピーして上から順に実行すれば通る形を保つ（コピペで詰まらせない）。
- 個人環境固有値（特定ユーザー名のパス等）を例として残さない。各自が変更すべき箇所は
  「各自の環境に合わせて変更」と明示する。
- 構築後に各ツールを使う導線（各リポジトリの clone → Claude 起動）を末尾で案内する。
